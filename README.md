# FontAwesome v5.7.0 - Icon set 

## Getting Start

### node
```cmd
    npm i font-awesome-iconset
```
```js
    // import with ESM
    import ICON_SET from 'font-awesome-iconset';

    // import with CommonJs
    const ICON_SET = require('font-awesome-iconset');
```

### CDN

```html
    <script src="https://cdn.jsdelivr.net/npm/font-awesome-iconset@5.7.0/dist/iconset.min.js"></script>

    <!-- ICON_SET global variable will be defined -->
```
